Web framework: Django
Frontend framework: Bootstrap
Python version: 3.5.2


Usage scenario
--------------
- First start the application (see Running application);
- Then open a browser and do GET request to:
    e.g. http://localhost:8000/


Running application
-------------------
- cd to project directory
- run command: 'python manage.py runserver'


Dependencies
------------
See 'requirements.txt'


Bootstrapping
-------------
- You need to have Python installed in your machine
- You may use virtual environment for installing dependencies. Global installation is fine too.
- Install dependencies by running 'pip install -r requirements.txt'.


Note
-----
* SQLite database file is included for viewing existing data.
* In case you are interested to add more data, login to Django admin site with following credential:
    username: superuser, password: pass1234
